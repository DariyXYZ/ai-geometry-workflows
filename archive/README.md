# Archive

This folder keeps historical reports, old presentations, one-off Rhino
experiments, and generated outputs that should not be part of the active work
surface.

Archived files are preserved for traceability, not for the normal AI read path.

## Folders

| Folder | Contents |
| --- | --- |
| `reports/` | Old public/team reports and roadmap snapshots |
| `rhino-experiments-2026-06/` | One-off Rhino scripts, PNGs, and JSON reports from earlier massing/modeling experiments |

## Rule

Do not add new active knowledge here. Promote useful lessons into `docs/`,
`decisions/`, or `NEWS.md`, then archive the raw artifact only if it may be
useful later.
